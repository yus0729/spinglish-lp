# App Store 審査用情報

## アプリ名
Spinglish

## サブタイトル
AI英会話練習 - 話して学ぶ英語

## 説明文
Spinglishは、AIを活用した英語スピーキング練習アプリです。

日本語のお題を見て英語で声に出す「瞬間英訳」、お手本音声に続いて声に出す「シャドーイング」、イラスト付きで覚える「英単語練習」、その日の出来事を英語で書く「英語日記」の4つのトレーニングで、英語が口から出る感覚を育てます。

【主な機能】
- 瞬間英訳：日本語のお題を英語で話すだけ。タイピング不要
- AI採点：伝わりやすさ・文法・自然さの3軸で即フィードバック
- 添削とネイティブ表現：模範解答 + ネイティブが使う自然な言い換えを提案
- シャドーイング：お手本音声を聴いて声に出す。AIが再現精度・発音を評価
- 英単語練習：イラスト付きで200語を直感的に習得（Advanced / Founders）
- 英語日記：その日の出来事を英語で書いてアウトプット練習
- 5カテゴリ（日常会話・旅行・ビジネス・技術・ニュース社会）
- 5レベル（超初級〜ネイティブ級）
- お気に入り保存で間違えた問題を復習
- セッション完了後のAI分析アドバイス
- 練習リマインダー通知

【プラン】
- Founders: Advancedの全機能を月額¥700で永続利用（最初の300人限定・解約後再加入不可）
- Standard: 1日5セッション、シャドーイングAI評価2回/日、英語日記1日1回（月額¥1,000 / 週額¥300、初回3日間無料）
- Advanced: 1日10セッション、シャドーイングAI評価4回/日、英単語練習200語、英語日記1日1回（月額¥1,500 / 週額¥400、初回3日間無料）

毎日少しずつ、英語で話す力を伸ばしましょう。
